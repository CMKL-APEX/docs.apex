## เริ่มต้น
สวัสดีและยินต้อนรับผู้ใช้งานทุกท่าน เข้าสู่โลกของ AI-HPC ใน Tutorial นี้จะพาทุกท่านให้รู้จักกับ รูปแบบทำงานของระบบ HPC โดยพื้นฐาน
และข้อจำกัดต่าง ๆ ที่ HPC ทำได้หรือไม่ได้ เมื่อเปรียบเทียมกับกับระบบคอมพิวเตอร์ปกติที่เราใช้งานกันทั่วไป   

### คอมพิวเตอร์ทั่วไปทำงานอย่างไร?
- หลักการทำงานของคอมพิวเตอร์ทั่วไป
- input / processing / memory / output
- อุปกรณ์ที่เกี่ยวข้อง
- ลักษณะการทำงาน
- ข้อจำกัดต่อการใช้งาน
  - scale up / scale out
  - price / performance

### HPC คืออะไร?
- หลักการทำงานของ HPC
- input / processing / memory / output
- อุปกรณ์ที่เกี่ยวข้อง
- ลักษณะการทำงาน
  - cluster
  - parallel
  - networking / interconnect
  - share storage
  - privileged access management (pam)
- ข้อจำกัดต่อการใช้งาน
  - scale up / scale out
  - price / performance

### HPC + AI = AI-HPC
- การใช้งานด้าน AI
- ประเภทตามลักษณะการใช้งาน
  - data processing
  - model training
  - model inferrencing
- ข้อแตกต่างกับ HPC ปกติ
  - การออกแบบที่เน้นด้าน AI โดยเฉพาะ
  - GPGPU (Nvidia / AMD / Intel / other card)
